## Monitoring Setup using Prometheus and Grafana

### Objective
Implement a basic monitoring and visualization stack for cloud and
container-based workloads.

### Tools Used
- Prometheus
- Grafana
- Docker Compose

### What I Implemented
- Configured Prometheus to scrape metrics
- Deployed Grafana for visualization
- Ran monitoring stack using Docker Compose

### Real-World Use Case
Monitoring is critical for cloud operations teams to ensure system
availability, detect anomalies, and reduce MTTR.

### Key Learnings
- Metrics-based monitoring
- Observability fundamentals
- Dashboard-driven troubleshooting
